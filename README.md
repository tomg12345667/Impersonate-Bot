# Discord Impersonation Bot 

This is a quick bot i made where you can run /impersonate user message and it will

- Create a webhook with the user to impersonate (so same profile pic and name)
- Send the impersonated message
- Delete the webhook

# To Run

Rename .env.example to .env and set the bot token and app id (and optionally change portal port)

then

```npm install```

then 

```npm start``` 
to execute
